# AI Powered FRA Atlas

This project demonstrates a simple WebGIS based FRA Atlas.

It visualizes sample Forest Rights Act (FRA) claim locations on a map.

States covered:
- Madhya Pradesh
- Odisha
- Telangana
- Tripura

Technologies Used:
- HTML
- JavaScript
- Leaflet.js
- OpenStreetMap

Future Improvements:
- AI based claim prediction
- Satellite forest monitoring
- FRA data analytics dashboard
